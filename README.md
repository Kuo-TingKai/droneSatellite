# 無人機蜂群對 LEO 衛星的電磁壓制模擬系統
## UAV Swarm Electronic Warfare Simulation System

這是一個完整的 Python 模擬框架，用於模擬無人機蜂群在平流層對低地球軌道 (LEO) 衛星群進行電磁壓制/訊號阻斷的技術。

## 專案結構

```
droneSatellite/
├── constants.py          # 物理常數和系統參數
├── utils.py              # 工具函數（座標轉換等）
├── channel.py            # 電磁波傳播模型
├── ground_terminal.py    # 地面終端與 SINR 計算
├── satellite.py          # 衛星軌道計算 (SGP4)
├── uav_swarm.py          # 無人機蜂群與優化策略
├── simulation.py         # 主模擬循環
├── visualization.py      # 視覺化模塊
├── main.py              # 主執行文件
├── requirements.txt     # 依賴項
└── README.md            # 本文件
```

## 安裝與使用

```bash
pip install -r requirements.txt
```

```bash
python main.py
```

## 核心功能模塊

### 1. 物理環境與參數設定
- **衛星群 (LEO Satellites)**: 使用 SGP4 演算法計算動態軌道位置
- **無人機蜂群 (UAV Swarm)**: 在平流層 (20 km) 進行干擾
- **地面終端 (Ground Terminals)**: 分佈在台灣島上的接收點

### 2. 電磁波傳播模型
- **自由空間路徑損耗 (FSPL)**: 基於 Friis 公式
- **大氣層衰減**: 考慮衛星-地面和無人機-地面的不同衰減
- **鏈路預算計算**: 完整的功率預算分析

### 3. 通訊性能評估
- **SINR 計算**: 訊號、干擾與噪聲比
- **阻斷判斷**: 基於 SINR 閾值的通訊狀態評估
- **性能統計**: 平均 SINR、阻斷率等指標

### 4. 蜂群優化策略
- **位置優化**: 使用簡化的粒子群優化 (PSO) 策略
- **適應度函數**: 最大化被阻斷的地面終端數量
- **動態調整**: 根據環境變化調整無人機編隊

### 5. 視覺化
- **3D 空間視覺化**: 展示衛星、無人機和地面終端的空間關係
- **2D 性能分佈圖**: 在地圖上顯示 SINR 分佈
- **時間序列圖**: 追蹤平均 SINR 和阻斷率隨時間的變化

## 技術細節

### 數學模型

#### 自由空間路徑損耗
```
FSPL = 20 * log10(4 * π * d * f / c)
```

#### 鏈路預算
```
P_Rx = P_Tx + G_Tx + G_Rx - L_FSPL - L_Atm
```

#### SINR 計算
```
SINR = P_Rx / (J_total + N)
```

其中：
- `P_Rx`: 接收訊號功率
- `J_total`: 總干擾功率
- `N`: 系統熱噪聲功率

### 預設參數

- **衛星**: 高度 550 km, 發射功率 20 dBW (100W), 增益 30 dB
- **無人機**: 高度 20 km, 干擾功率 10 dBW (10W), 增益 25 dB
- **地面終端**: 接收增益 40 dB, 系統噪聲溫度 290 K
- **頻率**: 12 GHz (Ku-band)
- **SINR 閾值**: -5 dB

## 注意事項

1. **TLE 數據**: 範例中使用的是假設的 TLE 數據。實際應用時應使用真實的衛星 TLE 數據（可從 CelesTrak 等來源獲取）。

2. **計算複雜度**: 模擬時間和計算量取決於：
   - 衛星數量
   - 無人機數量
   - 地面終端數量
   - 時間步長和總模擬時間

3. **優化演算法**: 當前實現使用簡化的優化策略。如需更精確的結果，可以實現完整的 PSO 或遺傳演算法。
